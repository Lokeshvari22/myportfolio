## 🚢 Deployment

Deployed on **Vercel**. Any push to `main` automatically redeploys the site.

To deploy your own fork:
1. Fork this repo
2. Import into [vercel.com](https://vercel.com)
3. Set root directory to `public/`
4. Click **Deploy**
